LVL_01: VARIABLES

Static Typing in Godot 4

In GDScript, you can force a variable to lock down to a specific data type using a colon (:). This prevents you from making game-breaking mistakes later on.

Look at this correct setup where variables match their box types perfectly:

var player_name : String = "Killswitch"
var items_count : int = 15
var game_running : bool = true

If you tell Godot a box is an int (whole number), it will instantly crash if you try to put a String (text wrapped in quotes) inside it.

Target Script (Analyze Line 5 Error):
1 | extends Node
2 |
3 | var inventory_slots : int = 10
4 |
5 | func _ready():
6 |     inventory_slots = "An Old Key"
